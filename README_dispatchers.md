# TriGo Dispatcher Guide

This guide is for Dispatchers using the TriGo platform. It outlines how to monitor operations and manage triders in your TODA area.

## Key Features for Dispatchers

- **Real-time Dashboard**:
  - Live trider status monitoring (Online, Busy, Offline)
  - Queue management system
  - Active and pending ride tracking
  - Real-time statistics

- **Live Map Overview**:
  - Real-time locations of all active triders
  - Pickup and drop-off points for ongoing rides
  - Terminal locations and coverage area visualization
  - Interactive map with custom markers

- **Ride Management**:
  - View all pending ride requests
  - Track active rides in real-time
  - Monitor ride completion status
  - View ride history and statistics

- **Trider Management**:
  - View all triders in your TODA
  - Monitor trider online/offline status
  - Track trider queue positions
  - View trider activity history

- **Statistics and Monitoring**:
  - Number of online triders
  - Current queue length
  - Active rides count
  - Pending requests count


2. **Dashboard Overview**:
   - View the main statistics cards at the top
   - Monitor the trider queue in real-time
   - Track active and pending rides
   - Watch the live map for trider locations

3. **Managing Rides**:
   - View incoming ride requests in the requests panel
   - Monitor active rides in the active rides section
   - Track ride progress from request to completion
   - View ride details including:
     - Passenger information
     - Pickup/drop-off locations
     - Assigned trider
     - Current status

4. **Monitoring Triders**:
   - View online/offline status of all triders
   - Track queue positions in real-time
   - Monitor trider locations on the map
   - View trider activity history

## Tips

- Keep the dashboard open during your shift for real-time monitoring
- Use the map view to ensure proper coverage of your area
- Monitor the queue to ensure fair ride distribution
- Watch for any unusual patterns in ride requests or trider behavior

## Support

If you encounter any issues:
- Check your internet connection
- Refresh the dashboard if data seems stale
- Contact technical support for persistent problems
- Document any recurring issues for system improvement
