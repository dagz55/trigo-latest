# TriGo Passenger Guide

Welcome to TriGo! This guide helps you book tricycle rides in the Talon Kuatro area.

## Key Features for Passengers

- **Smart Location Services**:
  - Auto-detect your current location for pickup
  - Search for addresses within Talon Kuatro
  - Save frequently visited locations
  - Choose from popular destinations
  - Select specific terminal exits

- **Interactive Map**:
  - Visual selection of pickup and drop-off points
  - Real-time trider locations
  - Live ride tracking
  - Clear route visualization

- **Booking Features**:
  - Quick ride booking process
  - Accurate fare estimation
  - Multiple payment options
  - Ride status updates
  - Trip history

- **User Experience**:
  - Clean, modern interface
  - Dark mode support
  - Responsive design for all devices
  - Real-time notifications
  - Smooth animations

2. **Book a Ride**:
   - Set your pickup location:
     - Use current location
     - Search for an address
     - Select from saved places
     - Choose a popular spot
   
   - Choose your drop-off point:
     - Search for destination
     - Select a terminal exit
     - Pick from saved locations
     - Choose a popular destination

   - Review your booking:
     - Check the estimated fare
     - Verify locations
     - Select payment method
     - Confirm booking

3. **Track Your Ride**:
   - Watch for trider assignment
   - Track trider's approach
   - Monitor ride progress
   - View estimated arrival time

4. **After the Ride**:
   - Confirm completion
   - Rate your experience
   - Save new locations
   - View trip summary
